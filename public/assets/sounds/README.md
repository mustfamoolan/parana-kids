# ملفات الأصوات

## ملف الصوت المطلوب

يجب إضافة ملف صوتي باسم `notification.mp3` في هذا المجلد.

### المتطلبات:
- الاسم: `notification.mp3`
- الحجم: أقل من 100KB (مفضل)
- المدة: 1-2 ثانية
- التنسيق: MP3 أو WAV

### مصادر مجانية:
- [Freesound.org](https://freesound.org) - ابحث عن "notification sound"
- [Zapsplat](https://www.zapsplat.com) - أصوات مجانية
- [Mixkit](https://mixkit.co/free-sound-effects/notification/) - أصوات إشعارات مجانية

### ملاحظة:
إذا لم يتوفر ملف الصوت، سيستخدم النظام Web Audio API لإنشاء صوت بسيط تلقائياً.
