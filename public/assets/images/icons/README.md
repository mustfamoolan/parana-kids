# PWA Icons

هذا المجلد يحتوي على أيقونات PWA للتطبيق.

## الأيقونات المطلوبة

- `icon-192x192.png` - أيقونة 192x192 بكسل
- `icon-512x512.png` - أيقونة 512x512 بكسل

## ملاحظة

الأيقونات الحالية هي نسخ من `ParanaKids.png`. للحصول على أفضل نتائج، يُنصح بإنشاء أيقونات بأحجام محددة (192x192 و 512x512) من الصورة الأصلية.

يمكنك استخدام أي محرر صور (مثل Photoshop, GIMP, أو أدوات online) لإنشاء الأيقونات بأحجام مناسبة.
